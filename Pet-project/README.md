В данной работе я проводил анализ бразильского интернет магазина на датасете Kaggle.
Были проведены операции по:
1. Первичному анализу данных
2. Очистке данных
3. Объеденению данных в одну таблицу
4. Иммитация аналитики в SQL
5. Проведен общий анализ метрик, Динамика(по месяцам/дням недели/часам), регионов и городов,
   категорий товаров и выручке за них, связь опозданий с негативными отзывами,когортный анализ,
   RFM-сегментация
6. написанны выводы и рекомендации по устранению проблем
